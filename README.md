# Page Tools - Chrome Extension

Page Tools - это расширение для Google Chrome, которое добавляет на любую страницу две функции: панель с информацией о сайте и переводчик текста с английского на русский и обратно. Расширение работает без регистрации.

## Стек технологий

* **React** - UI компоненты
* **Webpack** - сборка проекта
* **MyMemory API** - бесплатный API для перевода текста
* Backend и база данных отсутствуют - расширение работает на стороне браузера

## Функциональность

* Инфо о странице: заголовок, язык и количество слов
* Переводчик: перевод текста EN -> RU и RU -> EN
* Закрытие всех панелей одной кнопкой в popup

## Запуск локально

1. Склонировать репозиторий

2. Установить зависимости
```
npm install
```
3. Запустить сборку
```
npm run build
```
После сборки загрузить папку проекта как распакованное расширение в `chrome://extensions` (требуется включить режим разработчика)

## Установка через релиз

Скачать готовый ZIP-архив можно в разделе [Releases](https://github.com/Herbaa/Chrome-extension/releases):

1. Распаковать архив
2. Открыть `chrome://extensions`
3. Включить **режим разработчика**
4. Нажать **"Загрузить распакованное"** и выбрать папку
